Project write-up

This Senior Design project is a website built to present team and program information in a clear, easy-to-navigate format. It was developed by me as a polished deliverable that could be used as a real informational site.

My goal was to create a clean user experience for visitors by organizing content into dedicated pages and making key sections easy to find at a glance.

What it is

  • Type: individual website development project
  • Frontend: HTML, CSS, JavaScript
  • Core pages: landing page, season/year information, coaches, sponsors, and videos
  • Purpose: communicate program details in a structured, visual format
  • Experience: multi-page navigation with consistent layout and branding

Highlights

  • Content-first design: each page was created around a specific audience need, making navigation straightforward for new visitors.
  • Clear page flow: users can move from the landing page to focused sections without confusion.
  • Consistent visual structure: shared styling and page patterns keep the site cohesive.
  • Individual effort: all development tasks were handled by me, including layout, content, styling, and testing.

How it works

The site is organized into dedicated pages that present specific content areas while keeping navigation and styling consistent throughout.

  • The landing page introduces the project and provides quick access to the main sections.
  • Season/year content organizes information by timeframe for easier browsing.
  • The coaches page highlights staff and leadership roles.
  • The sponsors page recognizes supporting organizations and partners.
  • The videos page provides media content to showcase activity and engagement.

This structure kept development manageable and allowed me to work independently while maintaining a consistent final product.

What I would improve next

  • Mobile optimization: further refine spacing and layout behavior across all device sizes.
  • Content management: make updates easier with a lightweight CMS or structured data source.
  • Accessibility: expand keyboard support, contrast checks, and semantic labels.
  • Media optimization: compress and optimize images/videos for faster page loads.
